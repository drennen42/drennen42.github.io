Why JavaScript Is Taking Over

Javascript is the bomb for these reasons

JavaScript is a top language due to its ability to deliver rich web content and its relatively light weight and ease of use. JavaScript is a good fit for mobile development platforms; after all, both Apple and Google exposed JavaScript application programming interfaces to their mobile development toolkits in order to expand their developer base.1

JavaScript is the reason web users are able to input information and interact with most websites they visit each day; this is huge, it's the only language in the browser. JavaScript and its relationship to the browser are critical to our web user experience. Learning the language can be huge for a future computer programmer.4

So this is all great. JavaScript is easy (relatively speaking) to use, speedy, and hey, everyone's doing it! But, there are some limitations. The downside of an interpreted language is that testing can be optional in some people's minds - the compiler isn't there say no. Because there's no compiler you might be tempted to skip testing...or really skimp on it.5

Think about Google Maps, AirBnB, Uber, just about any hot application with a sexy client-side interface. That's JavaScript, and it's the only game in town. So, get busy learnin'!
