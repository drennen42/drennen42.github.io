Ruby Methods

Array#include?
is a public Ruby method that searches through all elements of an array for a given object, provided as a parameter by the object that called it, and returns the boolean 'true' or 'false' if that object is found in the array.

The method 'Array#include?' is a must-know for all Ruby developers. In fact, any method that searches for a given element in an array is a necessity for all programmers, regardless of the language they develop in.

In the vast majority of developer positions include large quantities of data that must be searched through. Many times the data is company data or client data that may include things such as first names, last names, addresses, phone numbers, etc etc. This data can be stored in an array or, more likely, many many arrays and possibly many arrays OF arrays. Without having a method that is capable of quickly scanning through all of that data, it would be nearly impossible to effectively and efficiently find and use any specific information.

Syntax: array_name.include?(element_you_need_to_find)

For example, if you have an array called 'clients' and it currently holds:
["Alex Drennen", "Fictional Person", "Michael Jordan", "Donald Trump"]

You can use this method to find out if I am a client with:
clients.include?("Alex Drennen")

This will return 'true'.

What may be even more important is that you can use this method within a loop and have all elements that are found in the array returned to be used or printed out. For example:

>>clients = ["Alex Drennen", "Fictional Person", "Michael Jordan", "Donald Trump"]
>>list_of_names = ["Bob McGee", "Alex Drennen", "Person Three", "George Fakerson", "Michael Jordan", "Chevy Chase"]
>> list_of_names.eah do |x|
>> if clients.include? (x)
>> puts x
>> end
>> end

This will print to the console all names that are in both the clients array and the list_of_names array. Which in this case would be:
"Alex Drennen"
"Michael Jordan"
